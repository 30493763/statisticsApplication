# Statistics Application

A console-based C# application that calculates various statistical measures for a given dataset of integers.

<img src="https://github.com/30493763/statisticsApplication/blob/master/statisticsApplication/image/Picture2.png" alt="" align="center" width="auto" height="auto">


## 📋 Description

This application allows users to input a series of integers and calculates the following statistical measures:
- **Mean (Average)**: The arithmetic average of all values
- **Median**: The middle value when data is sorted
- **Mode**: The most frequently occurring value(s)
- **Range**: The difference between maximum and minimum values
- **Standard Deviation**: Measure of data dispersion from the mean
- **Minimum**: The smallest value in the dataset
- **Maximum**: The largest value in the dataset

## ✨ Features

- Interactive console interface
- Dynamic data input (enter as many numbers as needed)
- Input validation (only accepts positive integers)
- Handles multiple modes (multimodal datasets)
- Detects when no mode exists (all values appear equally)
- Option to run multiple calculations without restarting
- Clear and formatted statistical output

## 🛠️ Technologies Used

- **Language**: C# 7.3
- **Framework**: .NET Framework 4.7.2
- **IDE**: Visual Studio
- **Data Structures**: List<int>, Dictionary<int, int>
